# Task Manager

אפליקציית ניהול משימות וצוותים מודרנית הבנויה עם Angular 21 ו-Angular Material.

## 🚀 התחלה מהירה

### דרישות מוקדמות
- Node.js 18+
- npm 11+

### התקנה
```bash
npm install
```

### הרצת האפליקציה
```bash
npm start
```



## 🔧 פקודות שימושיות

- `npm start` - הרצת שרת הפיתוח
- `npm build` - בנייה לייצור
- `npm test` - הרצת בדיקות
- `npm watch` - בנייה במצב צפייה

## 📋 תכונות

- 🔐 ניהול משתמשים עם התחברות והרשמה
- 👥 ניהול צוותים
- 📁 ניהול פרויקטים
- ✅ ניהול משימות
- 💬 מערכת הערות

## 🏗️ ארכיטקטורה

```
src/app/
├── components/     # קומפוננטות ה-UI
├── services/       # שירותים (API, auth, וכו')
├── guards/         # Route guards
├── interceptors/   # HTTP interceptors
├── models/         # סוגי TypeScript
└── app.routes.ts   # ניתוב הראשי
```

## 🛠️ טכנולוגיות

- **Angular 21** - Framework הראשי
- **Angular Material** - ספריית UI
- **TypeScript 5.9** - שפת תכנות
- **RxJS** - תכנות reactive
- **Vitest** - בדיקות יחידה
